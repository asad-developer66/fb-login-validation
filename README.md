# FB Login Validation

**Description:**  
This project is a Facebook login page clone with client-side form validation using HTML, CSS, and JavaScript. It demonstrates input validation, alerts, and page redirection.

---

## Features

- Email and password input validation  
- Minimum password length check (8 characters)  
- SweetAlert2 notifications for success and error messages  
- Redirects users to a specified URL after successful login
